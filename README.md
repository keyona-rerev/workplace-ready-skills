# Workplace Ready Skills Website

A clean, Apple-like website showcasing three immersive programs that develop practical workplace competencies.

## Site Structure

### Pages Included:
1. **index.html** - Homepage with overview and value proposition
2. **research-skills.html** - Internet Research Skills program page
3. **business-literacy.html** - Business Model Literacy program page
4. **professional-composition.html** - Professional Composition program page
5. **about.html** - About page (with placeholder content for team)
6. **contact.html** - Contact page with form and information

### Supporting Files:
- **styles.css** - Complete styling with your blue-green color palette
- **script.js** - Interactive functionality (mobile menu, form handling, animations)

## Color Palette Used

The site uses your Swatch Collection 4 with primary colors:
- **Primary**: Peacock Blue (#00cf91) - CTAs and accents
- **Secondary**: Olympic Blue (#4f8fe6) - Secondary accents
- **Accent**: Cerulian Blue (#29bdad) - Hover states
- **Backgrounds**: Nile Blue (#bfffe6), Pale King's Blue (#abf5ed)
- **Dark Text**: Deep Slate Green (#0f261f)

## Features

✅ **Responsive Design** - Works on desktop, tablet, and mobile
✅ **Sticky Navigation** - Always accessible menu bar
✅ **Smooth Animations** - Fade-in effects on scroll
✅ **Mobile Menu** - Hamburger menu for small screens
✅ **Professional Polish** - Clean, Apple-like aesthetic

## Design Notes

- **Typography**: System font stack for native OS feel
- **Spacing**: Consistent rhythm with CSS variables
- **Hover Effects**: Subtle elevations and color transitions
- **Accessibility**: High contrast, readable font sizes
- **Performance**: Minimal dependencies, fast load times
